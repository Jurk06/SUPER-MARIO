# Super Mario JS Bros
A vanilla JavaScript platformer game inspired by Super Mario Bros.

## Features
- Pixel-art style platformer gameplay
- Multiple levels with increasing difficulty
- Score system
- Sound effects and music
- Responsive controls (Arrow Keys, WASD, or on-screen)

## How to Play
- **Arrow Keys / WASD**: Move left and right
- **W / Up Arrow / Space**: Jump
- **F**: Shoot fire

## Local Development
Simply open `index.html` in your browser to play the game.

## Deployment
This game is deployed on Hugging Face Spaces and runs in a Docker container.

Enjoy the game! 🎮
